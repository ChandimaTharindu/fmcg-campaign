# FMCG Campaign Service

Spring Boot REST API for Admin and Agent campaign journeys.

## Stack
- Java 17
- Spring Boot 3
- Spring Web + Validation + JPA
- H2 in-memory DB

## Run
```bash
mvn spring-boot:run
```

## Test
```bash
mvn test
```
